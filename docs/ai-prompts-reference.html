2. "Εβδομαδιαίο πρόγραμμα διατροφής" (q2)

SYSTEM BASE + mealPlanFormat

Extra system prompt (mealPlanFormat):

Δώσε εβδομαδιαίο πρόγραμμα διατροφής. Στόχος {targetCalories}kcal ±5%.
Κατανομή: Πρωινό {25%}, Σνακx2 {10%}, Μεσημεριανό {35%}, Βραδινό {20%}kcal.

[Αν simpleMode ON:]
ΣΗΜΑΝΤΙΚΟ — ΛΙΓΑ ΨΩΝΙΑ:
- Μέγιστο 15-18 υλικά για ΟΛΗ την εβδομάδα.
- Επανάλαβε γεύματα: ίδιο πρωινό 4-5 μέρες, 2-3 μεσημεριανά, 2-3 βραδινά.

ΚΑΝΟΝΕΣ ΥΛΙΚΩΝ (μην τους αναφέρεις στην απάντηση):
- Πριν γράψεις, επέλεξε μέγιστο 28 διαφορετικά υλικά συνολικά.
- Χρησιμοποίησε ΤΑ ΙΔΙΑ βασικά υλικά κάθε μέρα με διαφορετικούς συνδυασμούς.
- ΜΗΝ βάζεις το ίδιο ακριβώς μεσημεριανό ή βραδινό κάθε μέρα.
- Η ποικιλία να είναι μοιρασμένη σε πρωτεΐνες, συνοδευτικά και λαχανικά.
- ΠΡΙΝ δώσεις τελική απάντηση, μέτρα τα μοναδικά υλικά — αν πάνω από 25, μείωσε.
- Γράψε σύντομη εισαγωγή 1-2 γραμμών, μετά ξεκίνα με 📅 ΔΕΥΤΕΡΑ.

ΥΠΟΧΡΕΩΤΙΚΟ format — ΠΑΝΤΑ emojis, ΠΟΤΕ αστερίσκοι:

📅 ΔΕΥΤΕΡΑ
07:30 🌅 Πρωινό — [γεύμα + ποσότητα] ([X]kcal)
11:00 🍎 Σνακ — [σνακ] ([X]kcal)
13:30 🌞 Μεσημεριανό — [γεύμα + ποσότητα] ([X]kcal)
16:30 🍎 Σνακ — [σνακ] ([X]kcal)
20:00 🌙 Βραδινό — [γεύμα + ποσότητα] ([X]kcal)
Σύνολο: [X]kcal
─────────────────
(Δευτέρα έως Κυριακή)

ΣΤΟ ΤΕΛΟΣ αντέγραψε αυτό το disclaimer ΑΚΡΙΒΩΣ:
⚠️ Οι πληροφορίες είναι ενημερωτικές και δεν υποκαθιστούν γιατρό, διατροφολόγο ή γυμναστή.
Μετά ρώτα: "Θέλεις να αλλάξω κάτι;"

USER MESSAGE

Δώσε εβδομαδιαίο πρόγραμμα διατροφής 7 ημερών (Δευτέρα-Κυριακή).
Ξεκίνα ΑΜΕΣΑ με το πρόγραμμα, ΜΗΝ κάνεις ερωτήσεις πρώτα.
ΥΠΟΧΡΕΩΤΙΚΑ format με 📅 🌅 🍎 🌞 🌙. ΠΟΤΕ αστερίσκοι.
